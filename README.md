# Projeto SQL - Mini-mundo Escolar

Este repositório contém scripts SQL para criação e manipulação de dados de um banco de dados escolar, baseado em um modelo lógico com entidades como alunos, turmas, professores, disciplinas e responsáveis.

## Scripts incluídos

- `insert_data.sql`: povoamento inicial das tabelas principais.
- `select_queries.sql`: consultas com JOIN, WHERE, ORDER BY e LIMIT.
- `update_delete.sql`: atualizações e exclusões com condições.

## Ferramentas recomendadas

- MySQL Workbench ou PGAdmin
- Git para versionamento

## Execução

1. Crie o banco de dados e as tabelas conforme o modelo lógico.
2. Execute os scripts na ordem: INSERT → SELECT → UPDATE/DELETE.
3. Verifique os resultados e ajuste conforme necessário.
